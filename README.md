# Reassessment Submission

## Overview
This repository contains two iOS projects:
1. **PersonalityQuiz** — a multi‑quiz personality app with dynamic answer layouts and optional timed mode.
2. **ToDoList** — a dark‑themed task manager with categories, due dates, reminders, search, and sharing.

## Repo Structure
- `PersonalityQuiz` — PersonalityQuiz app project
- `ToDoList` — ToDoList app project

## Setup Instructions
1. Install Xcode (latest stable recommended).
2. Open one project at a time
3. Select the app target and a simulator or device.
4. Build and run.
5. Allow notifications if prompted (used for reminders in ToDoList and optional features in PersonalityQuiz).

## PersonalityQuiz Features
- Multiple quizzes selectable from the intro screen
- Questions and answers randomized each run
- Dynamic answer UI for single, multiple, and ranged questions
- Timed quiz mode with per‑question countdown
- Results screen with personality type and definition
- Local history of completed quizzes with time data for timed runs

## ToDoList Features
- Create, edit, and delete tasks
- Search by title and notes
- Categories with colored tags (work, personal, completed)
- Grouped or filtered task list by category
- Due date picker with overdue/upcoming highlighting
- Local reminder notifications
- Share task details using the system activity sheet
